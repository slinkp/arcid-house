A minimal (for now)drum machine. There is no objective except to groove.

-- USER MANUAL --

  • JOYSTICK: move cursor to select a control.
  • A: toggle the selected control
  • SPINNER: adjust the selected control.

Note that not everything is spinnable or toggleable. Just try stuff.
